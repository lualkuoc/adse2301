HTML Lists

A list is a collection of related items e.g. day of the week or ingredients in recipe. This page will illustrate the various types of lists we can create in a web page.
Ordered Lists

This is a collection of item in a particular order or steps that need to be followed or carried out in a given order. Some examples are given below.
Days in a Week:

    Sunday
    Monday
    Tuesday
    Wednesday
    Thursday
    Friday
    Saturday

HTML Anchors or Links

An HTML anchor or hyperlink is some text or graphic that is used to direct users to a different part of your webpage, website or another external website. They can also be used for telephone or email links.
